## Problem

Managing job applications across users requires authentication and access control.

## Solution

Built a backend API with secure authentication and role-based permissions.

## Features

- JWT-based authentication
- Role-based access control (RBAC)
- Secure CRUD APIs
- Database migrations

## Tech Stack

- Node.js
- Express
- PostgreSQL
- Sequelize
- JWT

## Design Highlights

- Clear separation of concerns
- Migration-based schema evolution
- Scalable RBAC model
